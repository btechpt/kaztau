# Kaztau

Kaztai is a simple cli app to send message telegram. Just call command on shell or execute via cron for create reminder notification.

### Instalation
```shell
pip install ....
```
export your telegram credential
```shell
export KAZTAU_TELEGRAM_API_ID=112233
export KAZTAU_TELEGRAM_API_HASH='11aaa22bb'
export KAZTAU_TELEGRAM_BOT_TOKEN='22bb33cc'
```
